funkcja pozwala na zdobycie nazwy pliku w zależności od wybranego numeru pliku

funkcja bierze wartosc int określająca numer pliku przypisany według schematu co w funkcji phone book list a następnie wydobywa nazwę tego pliku i zwraca go do tablicy znaków podanych jako pierwszy argument funkcji

Parameters
file_name- tablica znaków do której zostanie zapisana wybrana nazwa pliku
choice- wartość liczbowa wybranego pliku przypisanej na tej samej zasadzie co w funkcji phone_book_list
Returns
int zwraca 0 gdy funkcja wykona się poprawnie

◆ phone_book_list()

int phone_book_list ( )

funkcje te generuje listę plików dostępnych w folderze przeznaczonym na zapis książek telefonicznych

na samy na początku funkcja wyświetla your phone books: następnie z pomocą biblioteki dirent.h otwiera lokalizacje przeznaczona na zapisywanie książek telefonicznych następnie również z pomocą biblioteki dirent.h wypisuje listę nazw plików bez rozszerzenia wraz przypisaniem do nich wartości numerycznej umożliwiających późniejszy wybór książki do wybrania

Returns
int funkcja zwraca wartość 0 jeżeli nie masz żadnych dostępnych książek telefonicznych i 1- jeśli masz dostępne książki telefoniczne
